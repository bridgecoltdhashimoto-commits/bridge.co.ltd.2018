---

## このキットで完成するもの

15分の基本ルートを終えると、次の8点が手元に残ります。

- **今日やる1つ**
- **今日やらない3つ**
- **相談時3行メモ**
- **AIに聞く前の質問整理メモ**
- **送る / 送らない / 保留の最終判断**
- **残す証拠リスト**
- **今日しない接触行動**
- **今日の初動カード**

## 1トラブル＝1初動カード

このキットは、1つのトラブルにつき1枚の「今日の初動カード」を作るために使います。

- 相手が変わったら、新しいカードを作る
- 内容が変わったら、新しいカードを作る
- 日付や状況が大きく変わったら、新しいカードを作る
- 過去のカードは消さず、自分の初動を見返す記録として残す

## このキットを開くタイミング

- AIに聞く前
- 人に相談する前
- 相手へ送る前
- 証拠や履歴を消す前
- ブロックする前
- 電話・直接接触・SNS投稿の前
- 相手から新しい返信が来たとき
- 新しい証拠や記録を見つけたとき

## 15分完成ルート

時間がないときは、次の順番だけ進めます。詳しいチェックは後から戻れます。

- [ ] **0〜3分：3分緊急停止モード** — 送信・削除・接触を止める
- [ ] **3〜5分：Gate 1** — 送る / 送らない / 保留を決める
- [ ] **5〜7分：Gate 2** — 残す証拠リストを作る
- [ ] **7〜9分：Gate 3** — 今日しない接触行動を決める
- [ ] **9〜11分：Gate 4** — AIに聞く前の質問整理メモを作る
- [ ] **11〜13分：Gate 5** — 今日やる1つ・やらない3つを決める
- [ ] **13〜15分：今日の初動カード** — 結果を1枚へまとめる

## はじめに

このキットは、トラブルの答えや結論を出すものではありません。

次のことは提供しません。

- 法律に関する個別の結論
- 本人に代わって行う連絡や交渉
- 結果の約束
- 治療や医療上の結論
- 緊急時の安全確保
- 証拠そのものの保管

### 危険があるとき

次に当てはまる場合、このキットより安全確保を優先してください。

- 暴力や脅迫がある
- 自傷・他害のおそれがある
- けが、急病、身の危険がある
- 今すぐ保護や緊急対応が必要である

この場合は相手へ連絡せず、警察・救急・自治体などの公的窓口や、近くの信頼できる人へ連絡してください。

### このキットの役割

このキットは、答えを増やすものではありません。

AIに聞く前、人に相談する前、相手に連絡する前に、次の順番で使います。

1. 送る前に止まる
2. 消す前に止まる
3. 接触する前に止まる
4. AIに聞く前に質問を整える
5. 今日やることを1つ、やらないことを3つ決める

### 使い方

- Gate 1から順に進めます。
- すべてを埋める必要はありません。
- 判断できない項目は「保留」にします。
- 強い怒りや焦りがあるときは、送信や接触を止めるだけでも構いません。
- トラブルが変わるたびに、新しいコピーで使い直してください。

### 共通ルール：迷ったら保留

迷った項目は、無理に決めず**「保留」**にしてください。

保留は失敗ではありません。送る・消す・接触する前に止まれた時点で、このキットの役割は果たしています。

# 初動ミスで起きやすいこと

- 感情のまま送ると、後で説明しにくい文面が残ることがあります。
- 証拠を消すと、相談時に状況を説明しにくくなることがあります。
- ブロック前に記録を残さないと、後から確認できなくなることがあります。
- 電話や直接接触で、話がさらにこじれることがあります。
- SNS投稿で第三者に広がり、戻せなくなることがあります。
- AIに質問を丸投げすると、回答が増えてさらに混乱することがあります。

このキットは、結果を約束するものではありません。ただし、こうした初動ミスを減らすために、送る前・消す前・動く前の順番を固定します。

# Gate 1：送信ブレーキ

## 目的

LINE・DM・メールを送る前に、感情的な文面、攻撃に見える文面、後から見返されて困る文面を止めます。

## 記入例

**悪い例**  
「なんで無視するんですか。逃げるなら全部晒します。」

**良い例**  
「確認したいことがあります。〇月〇日の件について、事実関係だけ確認させてください。」

> 良い例は文章の形を比べるための例です。送信を勧めるものではありません。迷ったら保留します。

## 1-1. 今送る必要がありますか

- [ ] 今日中に返答が必要だと確認できている
- [ ] 今送らないと安全や生活に直接影響する
- [ ] 相手から急かされているだけではない
- [ ] 明日まで待っても状況は大きく変わらない

**いまの結論**

- [ ] 今は送らない
- [ ] 10分以上置いてから見直す
- [ ] 短い確認だけ送る
- [ ] 先に第三者へ相談する

## 1-2. 送る目的を確認する

送る目的はどれですか。1つだけ選びます。

- [ ] 事実を確認する
- [ ] 日程や期限を確認する
- [ ] 受け取ったことだけ伝える
- [ ] 自分の希望を短く伝える
- [ ] 怒りをぶつける、責める、反応させる
- [ ] 自分でも決まっていない

「怒りをぶつける」「自分でも決まっていない」を選んだ場合、今日は送信を保留します。

## 1-3. 文面の一時停止チェック

- [ ] 相手を脅すように読める表現がない
- [ ] 怒り、恨み、皮肉、決めつけを入れていない
- [ ] 相手の人格を責めていない
- [ ] 事実と自分の推測を分けている
- [ ] 第三者に見られても困らない
- [ ] 後日、自分で読み返しても説明できる
- [ ] 深夜や飲酒後の勢いで作っていない
- [ ] 個人情報や不要な秘密を書いていない

1つでも迷う場合は送信を保留します。

## 1-4. 短い確認だけにするメモ

文章を完成させる欄ではありません。送る目的と事実だけを残します。

# Gate 4：AI相談前フィルター

## 目的

AIに聞く前に、事実・感情・推測・個人情報を分け、今日知りたいことを1つに絞ります。

## AIに聞く前の重要な注意

このGateで作るものは、AIへそのまま投入して答えを決めさせるための入力文ではありません。

AIに聞く前に、事実・感情・推測・個人情報を分け、質問を1つに絞るための**AI相談前メモ**です。AIへ渡す前の前処理メモとして使います。

AIの回答は、間違っている・古い・状況に合わない可能性があります。

AIの回答をそのまま相手に送ったり、法律上・医療上・緊急時・金銭請求に関する判断や、相手への具体的対応の最終判断に使ったりしないでください。

### AIに入力しないもの

- 実名
- 住所
- 電話番号
- 口座番号
- パスワード
- 相手の個人情報
- 証拠画像そのもの
- 契約書や診断書などの重要文書全文
- 相手を脅す、責める、追い詰める目的の文章

### AI回答を使う前の確認

- [ ] 回答が自分の状況に合っているか確認する
- [ ] 法律・医療・緊急・金銭に関する判断は、公的窓口や適切な専門家へ確認する
- [ ] AIが作った文章をそのまま相手に送らない
- [ ] 送る前に**Gate 1：送信ブレーキ**へ戻る
- [ ] 証拠を扱う前に**Gate 2：証拠保全ブレーキ**へ戻る
- [ ] 電話・接触・投稿の前に**Gate 3：接触ブレーキ**へ戻る

# 最後のまとめ

このキットは、トラブルの結果を約束するものではありません。

それでも、次の初動ミスを減らすための順番として使えます。

- 感情のまま送る
- 記録を確認せず消す
- 勢いでブロックする
- 電話や直接接触を急ぐ
- SNSへ投稿する
- 質問が散らかったままAIへ入力する
- 全部を一度に進めようとする

トラブルのたびに、最初にこのキットを開いてください。

1. AIに聞く前
2. 人に相談する前
3. 相手に送る前
4. 消す・ブロックする前
5. 電話・接触・投稿の前

止まる、残す、整える、1つに絞る。  
それが「BRIDGE パニック・ファイアウォール mini」の役割です。

# 再利用方法

- 1つのトラブルにつき、原稿を1回コピーして使います。
- 相手・内容・日付が変わったら、新しいコピーで始めます。
- 記入済みのシートは、次に同じ初動ミスを避けるための記録になります。
- 新しい連絡や証拠が増えたときは、元の記入を消さず、日付を付けて追記します。
